Description

The GeoS project is a web tool that allows the user to view real-time images on the map.

Images will be taken from 500px, Flickr and Instagram - which include geo-tagging information.

The tool will provide support for the use of various viewing filters - modularly deployed - related to areas of interest (e.g. only Central Europe, only images taken in cities), attributes of photographs (eg, color, size, orientation, type, other IPTC / EXIF metadata etc.), date & time (for example, displaying images created in the last L months or only those in year A).

Ideas/Concepts on design

Since the main activity takes place at the map level, we have opted for a three-tab slide menu, one containing different filters to view the photos that meet certain criteria, one containing different application settings and one which will contain information about the functionality of the application.

We also included the option to search for certain regions on the map to make it easy to see the area of interest in detail.

User Interface

The user interface design is responsive and mobile-friendly.
